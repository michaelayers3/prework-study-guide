HTML

  • HTML is the structure of the website. It's like the foundation of a house.
  • The head element contains informations about the webpage.
  • Head elements will not show on the website.
  • The body element represents the visible content shwon to the user.

CSS

  • CSS is the design of the website, dealing with colors, margins, borders, padding, and more.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout -b pull origin main: Pulls current branch to main branch.

JavaScript

  • JavaScript is the interactivity of the website, allowing users to click or type on the website.
  • JavaScript uses conditional statements, like if and then, in order to respond to certain criteria inputted.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
